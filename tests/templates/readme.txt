Run several script to compute and print graphs with RDF using :
1) a config file : ~/.multirdfrc 
                   with default parameters for all runs
                   each row is one parameter
2) an input file : ./parameters.in
                   each row is oneset of parameters
                   first row provide parameters' name for each column

Put executable path into your current $PATH and run the main executable by :
$ ln -s /path/to/exec/multirdf /path/to/python/modules/

$ multirdf
